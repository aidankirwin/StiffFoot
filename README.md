# Optimizing Prosthetic Foot Stiffness to Minimize the Metabolic Cost of Walking

Optimization of prosthetic foot stiffness to minimize metabolic cost during below-knee amputee walking.

## Setup
1. Install Anaconda or Miniconda, Git 
2. Clone the repository
3. Install OpenSim GUI
4. Install the OpenSim Conda package (this includes Moco):
```bash
conda install -c opensim-org opensim
```
5. Install the 'requirements.txt' to install the packages
